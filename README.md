Personal Multi-Page Static Portfolio
-----------------------------------
Files:
- index.html (Home)
- about.html (About)
- projects.html (Projects)
- contact.html (Contact)
- styles.css (global styles)
- assets/ (images and favicon)

To preview locally:
- Open index.html in your browser
- Or run: python -m http.server 127.0.0.1:5500  (then visit http://localhost:127.0.0.1:5500)
